# FinLedger — Учёт доходов и расходов

Приложение для телефона (PWA) для учёта личных финансов с аналитикой и рекомендациями.

## Установка на телефон

### Способ 1: Установить как приложение (рекомендуется)

1. Откройте в браузере телефона: **[finledger-app.netlify.app](https://finledger-app.netlify.app)**
2. Походите по вкладкам 30 секунд (Главная → Операции → Аналитика)
3. Внизу экрана появится баннер «Добавить на главный экран» — нажмите
4. **Андроид (Chrome):** меню ⋮ → «Установить приложение»
5. **iPhone (Safari):** кнопка «Поделиться» → «На экран Домой»

После установки приложение работает как обычное — со своей иконкой и без строки браузера.

### Способ 2: Просто открыть в браузере

Перейдите по ссылке выше — всё работает без установки, данные сохраняются в телефоне.

### Способ 3: Запустить локально

Скачайте файлы из этого репозитория, откройте `index.html` в браузере или запустите локальный сервер:
```
python -m http.server 8080
```
Затем откройте http://localhost:8080

## Возможности

- Учёт доходов и расходов
- Категории (12 встроенных + свои)
- Аналитика: расходы по категориям, средний чек, процент сбережений
- Персональные рекомендации по управлению бюджетом
- Импорт/экспорт данных
- Тёмная тема, современный дизайн
- Работает без интернета (после первой загрузки)

## Перенос данных между устройствами

Настройки → Экспорт данных → сохраните файл → на другом устройстве Настройки → Импорт данных.

## Технологии

- HTML5 + CSS3 + JavaScript (ванильный, без фреймворков)
- PWA (Service Worker, Web App Manifest)
- Локальное хранение (localStorage)
- Canvas API для графиков
